# 镜像烧录指南

> 更新日期：2026-07-04

---

## 一、前置说明

出厂已预置测试镜像，正式开发需将系统烧录至 eMMC，以确保 luckfox-config、ADB 等功能正常。

进入烧录模式的两种方式：
- **硬件触发**：按住 BOOT 键再连接 USB
- **软件触发**（已通过 ADB 连接时）：`adb shell reboot loader`

---

## 二、Windows 烧录（SocToolKit v1.98）

### 安装驱动

1. 下载 DriverAssitant v5.13
2. 以管理员身份运行，安装 USB 驱动
3. 安装完成后**重启电脑**

### 完整镜像一键烧录

1. 按住 BOOT 键同时连接电脑，等待工具显示 `MaskRom` 后松开
2. 点击 **Firmware** 选择固件目录
3. 点击 **Upgrade**

### 分区选择性烧录

1. 同上进入 MaskRom 模式
2. 点击 **Search Path** 选择目录
3. 勾选需要烧录的分区
4. 点击 **Download**

## 三、Linux 烧录（Ubuntu 22.04 x86_64）

### 安装 upgrade_tool

```bash
sudo unzip upgrade_tool_v2.17.zip
cd upgrade_tool_v2.17_for_linux/
sudo cp upgrade_tool /usr/local/bin
sudo chmod +x /usr/local/bin/upgrade_tool

# 验证
sudo upgrade_tool -v
# 输出：Upgrade Tool v2.17
```

### 烧录固件

```bash
# 直接烧录
sudo upgrade_tool uf update.img

# SDK 一键脚本（需在 output/image/ 目录下有 update.img）
sudo ./rkflash.sh update
```

---

## 五、注意事项

- 镜像文件名末六位为日期，**务必选择最新版本**
- Linux 版 upgrade_tool 仅支持烧录至 SPI NAND Flash 或 eMMC
- 虚拟机无法识别设备时：检查数据线是否为数据线（非充电线），或重新插拔让 VMware 重新分配控制权
