## 一、CSI 摄像头

### 1.1 硬件连接

- 打开 CSI 排线黑色卡扣（双侧均匀用力）
- 插入排线，**金属面朝向开发板背面**
- 扣紧卡扣

支持摄像头型号：
| 型号 | 规格 |
|------|------|
| GC2093 | 200 万像素，当前使用 |
| ISG1321 | 138 万像素，全局快门 |
| GC2093 | 200 万像素，备选型号 |

### 1.2 系统初始化

烧录官方镜像并重启一次，RKIPC 自动切换到对应摄像头配置：

```bash
ls /userdata/
# 应能看到 rkipc.ini 文件生成，说明摄像头识别成功
```

### 1.3 RTSP 推流测试

获取设备 IP：

```bash
ifconfig
# USB 静态 IP：172.32.0.93
```

VLC 拉流地址：

```
rtsp://172.32.0.93/live/0
```

> 建议将 VLC 缓存时间降低到 300~500ms，平衡延迟与稳定性（默认 1000ms 延迟较高）。

### 1.4 UVC 模式

将开发板作为标准 USB 摄像头使用：

```bash
# SDK 配置中添加
export RK_APP_TYPE="RKIPC_RV1106 UVC_TINY"

# 板端执行
RkLunch-stop.sh
killall S99usb0config   # 防止 adb 重连干扰
usb_config.sh
rk_mpi_uvc &
```

之后在 PC 端用 PotPlayer（`Ctrl+J`）或 OBS 识别摄像头设备。

## 二、Wi-Fi 配置

> Luckfox Pico Zero 内置 2.4GHz WiFi6 模块（RV1106G3）

### 2.1 连接 Wi-Fi

**编辑配置文件：**

```bash
nano /etc/wpa_supplicant.conf
```

写入内容：

```
ctrl_interface=/var/run/wpa_supplicant
ap_scan=1
update_config=1
network={
    ssid="你的WiFi名称"
    psk="你的WiFi密码"
    key_mgmt=WPA-PSK
}
```

**重启并获取 IP：**

```bash
killall -9 wpa_supplicant
wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant.conf
udhcpc -i wlan0
```

### 2.2 开机自动连接

创建 `/etc/init.d/S99wlan0`：

```sh
#!/bin/sh
case $1 in
    start)
        killall -9 wpa_supplicant
        wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant.conf
        udhcpc -i wlan0
        ;;
    stop)
        killall wpa_supplicant
        ;;
esac
```

赋权后重启：

```bash
chmod +x /etc/init.d/S99wlan0
reboot
```

### 2.3 切换板载/外接天线（可选）

```bash
echo 123 > /sys/class/gpio/export
echo out > /sys/class/gpio/gpio123/direction
echo 1 > /sys/class/gpio/gpio123/value   # 1=外接天线，0=板载天线
```

## 三、蓝牙配置

> Bluetooth 5.2 / BLE

### 3.1 开发板连接蓝牙耳机（A2DP Source）

```bash
bluealsa -p a2dp-source &
bluetoothctl
```

在 bluetoothctl 交互界面中：

```
power on
pairable on
scan on
# 等待目标设备 MAC 地址出现后
scan off
pair <MAC地址>
```

播放音频：

```bash
aplay -D bluealsa:DEV=<MAC地址>,PROFILE=a2dp audio.wav
```

### 3.2 手机连接开发板播放（A2DP Sink）

```bash
killall bluetoothd
/usr/libexec/bluetooth/bluetoothd --compat &
bluealsa -p a2dp-sink &
bluealsa-aplay -D hw:0,0 --profile-a2dp &
bluetoothctl
```

在 bluetoothctl 中：

```
power on
pairable on
discoverable on
```

手机搜索到设备（默认名 `BlueZ 5.65`）后点击配对，开发板终端输入 `yes` 确认配对。

## 四、GPIO 引脚

Luckfox Pico Zero 引出 28 个引脚，包含以下功能引脚：

| 功能 | 接口 |
|------|------|
| UART 调试串口 | TX / RX / GND |
| MIPI CSI | 4-lane，摄像头接口 |
| USB 2.0 | Host / Device |
| I2C | ≤5 路 |
| SPI | ≤2 路 |
| PWM | ≤12 路 |
| ADC | ≤2 路，10bit |
| 以太网 | 10/100M + 内嵌 PHY |

> 完整引脚图请参考官方 Wiki：[外设与接口](https://wiki.luckfox.com/zh/Luckfox-Pico-Zero/Pinout)
